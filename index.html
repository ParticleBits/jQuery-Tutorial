jQuery Tutorial Plugin

This plugin displays a very clean and simple tutorial flow for any web page. All you need to do is set up your markup correctly and you're good to go. To initialize it, just call the following on your tutorial element in your HTML: $('#tutorial').tutorial();

Include the Script and Stylesheet

There's 1 stylesheet and 1 javascript file that you need to include on any page displaying the tutorial. Also, please copy the images folder 'tutorial' to the same directory that your jquery.tutorial.css file is in. If you would like to move the location of these images, then please update the jquery.tutorial.css image paths appropriately. 

<link href="css/jquery.tutorial.css" rel="stylesheet" type="text/css" media="screen" />
<script src="js/lib/jquery.1.6.2.min.js" type="text/javascript"></script>
<script src="js/jquery.tutorial.js" type="text/javascript"></script>

Configuring the Tutorial

You have a few options when configuring the tutorial. To start the tutorial, you need to call $('#tutorial').tutorial(); anywhere in your application. You can optionally pass in an options object to this function by calling: 

$('#tutorial').tutorial( options );

These are the following options you can set:

  • location

    Default: 'tl'

    Any one of the valid window location values mentioned below.

  • width

    Default: 320

    Integer, in pixels, for the width of the tutorial window.

  • opacity

    Default: 0.4

    Decimal between 0 and 1.0 for the overlay opacity. 1.0 is opaque.

  • returnTop

    Default: true

    Boolean signaling whether or not the tutorial should return the user back to the top of the page when complete.

Structure of the Tutorial Element

The tutorial elementis simply a div with class tutorial containing a set of slide elements. To start, add this code to your page: 

<div id="tutorial" class="tutorial">
    <!-- slides will go here -->
</div>

The tutorial element contains a collection of slide divs with the following elements and HTML5 data properties: 

<div data-target="#selector .for:target" data-arrow="tc" data-location="tl">
    <h1>Headline for Step 1</h1>
    <p>
        This is the message for Step 1.
    </p>
</div>

Each step div requires both the data-target and data-arrow properties to be set. The data-location defaults to the options you specify when creating the tutorial. However, if you'd like to move the location of the tutorial window for a certain step, set the data-location to one of 'tl', 'tr', 'bl', or 'br' for top left, tor right, bottom left, and bottom right respectively. You need to set it back in the next step if you change the window location mid-tutorial.

Data arrow takes one of 'tc', 'ml', 'mr', or 'bc' for top center, middle left, middle right, or bottom center respectively. You can mess around with these to see how they interact or take a look at the tutorial on this page by clicking "Start Demo" at the top of the page.

Setting the Target, Arrow, and Window Location

Target

The data-target property needs to be any valid, jQuery selector that targets one element on the page. It could be an ID (preferred) or a selector choosing the first child, nth element, or anything at all selecting one element. If you choose a selector with more than one it will default to the first element of that collection.

Arrow Location

Arrow locations can be any of the following options:

  • tc Top center
  • ml Middle left
  • mr Middle right
  • bc Bottom center

Window Location

Window location can be any of the following options:

  • tl Top left
  • tr Top right
  • bl Bottom left
  • br Bottom right

It is important to note that if you change the window location mid-way through the tutorial, the window location will remain at the new location until you change it back.

Showing the Tutorial and Calling the Controls Programmatically

You have the ability to start the tutorial, move next or previous in the slides, or cancel the tutorial in your application. To start the tutorial simply call $('#tutorial').tutorial();. #tutorial is the ID of the element containing all of your slides. It needs to have class tutorial to work properly.

Next Slide

To move to the next slide, call the following on the same element you initialized the tutorial with: 

$('#tutorial').tutorialNext();

Previous Slide

To move to the previous slide, call the following on the same element you initialized the tutorial with: 

$('#tutorial').tutorialPrev();

Cancel Tutorial

To cancel the slideshow, call the following on the same element you initialized the tutorial with: 

$('#tutorial').tutorialCancel();
